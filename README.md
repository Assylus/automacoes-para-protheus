# Automações para Protheus

Repositório de scripts e automações para o sistema **TOTVS Protheus**.

## Estrutura

```
automações-para-protheus/
├── README.md
└── scripts/        # Scripts de automação (em breve)
```

## Como usar

1. Clone o repositório
2. Navegue até a pasta do script desejado
3. Siga as instruções de cada automação
